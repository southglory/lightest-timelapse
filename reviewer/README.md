# Reviewer — 타임랩스 캡처 검수 도구

캡처 이미지를 빠르게 훑고, 민감 정보를 가리고, 내보내는 도구.

## 빠른 시작

```bash
# 특정 세션
python -m reviewer D:/timelapse/2026-03-20_14-30-00

# 최근 세션
python -m reviewer latest
```

## 워크플로우

### 1. 훑어보기 + 삭제

그리드에서 WASD/화살표로 이동, 불필요한 이미지는 Delete.

### 2. 템플릿 만들기

반복되는 위치(작업표시줄, 시계 등)를 한번만 편집하면 전체에 적용.

```
뷰어 → 사이드바 [제작] → 레이어 추가 + 영역 편집 → [저장 완료]
```

### 3. 템플릿 적용

```
뷰어: 체크박스 토글로 현재 이미지에 즉시 적용/해제
      [전체적용] → 세션 내 모든 이미지에 일괄 적용

그리드: 이미지 선택 → [템플릿 적용] → 선택 이미지에 일괄 적용
```

### 4. 개별 편집

도구 선택 (1/2/3/4) → 영역 드래그 → 자동 저장.

### 5. 내보내기

그리드에서 [내보내기] → `exported/` 폴더에 원본+편집 합성 이미지 저장.

| Delete | 선택 삭제 | 이미지 삭제 |
| 1 | — | 모자이크 |
| 2 | — | 블러 |
| 3 | — | 가리기 |
| 4 | — | 펜 |
| Ctrl+Z | — | 실행취소 |
| Ctrl+Y | — | 재실행 |
| Ctrl+S | — | 수동 저장 |
| Escape | — | 도구 해제 / 그리드 복귀 |

## 데이터 구조

원본 이미지는 절대 수정하지 않습니다.

```
세션폴더/
├── *.jpg                  ← 원본 (불변)
├── .edits/                ← 이미지별 편집 데이터
│   └── 14-30-15.json     ← {"edits": [...], "applied_templates": [...]}
├── .templates/            ← 템플릿 (레이어 구조)
│   └── 하단바가림.json
├── .deleted/              ← 소프트 딜리트 (복구 가능)
├── .thumbs/               ← 썸네일 캐시
└── exported/              ← 내보내기 결과
```

## 템플릿 참조 모델

이미지에 템플릿 편집을 합치지 않고 **이름만 참조**.

- 적용: `applied_templates`에 이름 추가
- 해제: 이름 제거
- 템플릿 수정 시: 적용된 모든 이미지에 자동 반영
