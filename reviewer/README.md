# Reviewer — 타임랩스 캡처 검수 도구

타임랩스로 캡처한 이미지를 빠르게 훑어보고, 민감 정보를 가리고, 타임랩스 영상으로 만드는 올인원 도구.

---

## 빠른 시작

### exe 사용 (권장)

[Releases](https://github.com/southglory/lightest-timelapse/releases/latest)에서 `reviewer.exe`를 받으세요.

```
reviewer.exe D:\timelapse\2026-03-20_14-30-00
```

### Python으로 실행

```bash
pip install Pillow

# 특정 세션
python -m reviewer D:\timelapse\2026-03-20_14-30-00

# 가장 최근 세션
python -m reviewer latest
```

## 워크플로우

```
 ① 훑어보기        ② 템플릿 제작       ③ 일괄 적용        ④ 개별 편집        ⑤ 내보내기 + 영상
┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ 그리드에서 │    │ 뷰어에서   │    │ 체크박스   │    │ 도구 선택  │    │ [내보내기] │
│ WASD 이동  │ →  │ [제작] 클릭│ →  │ 토글로    │ →  │ 후 드래그  │ →  │ [영상생성] │
│ Delete 삭제│    │ 레이어 편집│    │ 즉시 적용  │    │ 자동 저장  │    │           │
└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
```

### ① 훑어보기 + 삭제

그리드에서 WASD/화살표로 이미지를 넘기며 확인합니다.
불필요한 이미지는 Delete로 삭제 (소프트 딜리트, 복구 가능).

- **이동**: ↑↓←→ / WASD
- **열기**: Enter 또는 더블클릭
- **선택**: Space (토글), Ctrl+클릭 (다중), Ctrl+A (전체)
- **삭제**: Delete

### ② 템플릿 제작

작업표시줄, 시계 등 **매번 같은 위치**를 가려야 할 때 사용합니다.
한번만 만들면 모든 이미지에 적용할 수 있습니다.

1. 뷰어에서 아무 이미지를 열기
2. 사이드바 **[제작]** 클릭 → 배경이 유리(glass) 효과로 변함
3. 레이어 추가 (+) → 이름 입력
4. 도구 선택 (1~4) → 가릴 영역 드래그
5. **[저장 완료]** → 이름 입력 → 저장

### ③ 템플릿 적용/해제

**뷰어에서 (현재 이미지):**
- 사이드바 상단에 모든 템플릿이 체크박스로 나열
- 체크 = 적용, 해제 = 체크 해제. 원클릭.
- [전체적용] → 모든 이미지에 일괄 적용 (비동기)
- [전체해제] → 모든 이미지에서 일괄 해제

**그리드에서 (다중 이미지):**
- 이미지 선택 → [템플릿 적용] → 체크 → 적용

### ④ 개별 편집

특정 이미지에만 추가 편집이 필요할 때:

1. 도구 선택 (1: 모자이크, 2: 블러, 3: 가리기, 4: 펜)
2. 영역 드래그 → 즉시 적용
3. 다음 이미지로 넘기면 자동 저장

**편집 요소 조작 (PPT처럼):**
- 도구 미선택 상태에서 편집 클릭 → 선택 (파란 테두리 + 핸들)
- 드래그 → 이동
- 모서리 핸들 드래그 → 리사이즈
- Delete → 선택된 편집 삭제

### ⑤ 내보내기 + 영상 생성

1. **[내보내기]** → `exported/` 폴더에 편집 적용된 이미지 저장
2. **[영상 생성]** → 내보내기된 이미지 또는 원본으로 MP4 영상 생성
   - FPS 설정 가능 (기본 30)
   - ffmpeg 내장 (별도 설치 불필요)

## 데이터 구조

원본 이미지는 **절대 수정하지 않습니다.** 모든 편집은 별도 파일로 관리됩니다.

```
세션폴더/
├── *.jpg                  ← 원본 (불변)
├── .edits/                ← 이미지별 편집 데이터
│   └── 14-30-15.json
├── .templates/            ← 템플릿 (레이어 구조)
│   └── 하단바가림.json
├── .deleted/              ← 소프트 딜리트 (복구 가능)
├── .thumbs/               ← 썸네일 캐시
├── exported/              ← 내보내기 결과
└── 세션이름.mp4           ← 생성된 영상
```

### 템플릿 참조 모델

이미지에 템플릿 편집을 합치지 않고 **이름만 참조**합니다.

```json
// .edits/14-30-15.json
{
  "edits": [...],
  "applied_templates": ["하단바가림", "시계모자이크"]
}
```

- **적용**: 이름 추가 → 렌더링 시 템플릿 내용 참조
- **해제**: 이름 제거 → 원본 상태로 복원
- **템플릿 수정**: 적용된 모든 이미지에 자동 반영
